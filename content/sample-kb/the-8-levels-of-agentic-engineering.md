| Level | Name | Description |
|-------|------|-------------|
| 1 | Tab Complete | Copilot-style autocomplete suggestions |
| 2 | Agent IDE | AI-focused IDEs (e.g. Cursor) with multi-file editing |
| 3 | Context Engineering | Optimizing information density in prompts and tools |
| 4 | Compounding Engineering | Plan-delegate-assess-codify loop with accumulated rules |
| 5 | MCP and Skills | Extending LLMs via MCP servers and custom skills |
| 6 | Harness Engineering | Observability, testing, and automated feedback loops |
| 7 | Background Agents | Async parallel agents with orchestrators |
| 8 | Autonomous Agent Teams | Agents coordinating directly, without a central hub |
