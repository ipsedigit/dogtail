Per iniziare un progetto con Claude Code assicurati che qualcuno stia pagando per te un account ad Anthropic. Scarica Claude Code (non ti spiego come), loggati, poi apri una shell, crea la directory del tuo progetto, entraci e lancia il comando claude. Poi lancia /init e accetta che crei il file CLAUDE.md
Ci sono tre CLAUDE.md diversi:
* quello di progetto, che viene pushato
* quello CLAUDE.local.md, che non viene pushato e contiene istruzioni locali e personalizzazioni
* .claude/CLAUDE.md nella tua home, condiviso tra tutti i tuoi progetti
